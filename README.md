# seven-scraping
Web Scraping for 7-eleven and Visualisation

## Requirement
curl -O https://raw.githubusercontent.com/apisit/thailand.json/master/thailandWithName.json

## How to run
```
pip install -r requirements.txt
python scraper/seven_scraper.py
```
